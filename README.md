# Medical Appointment Booking

## 📌 Project Overview

The **Medical Appointment Booking** website is an online platform that allows users to conveniently schedule appointments with doctors across various medical specialties. It aims to simplify the healthcare booking process by providing an easy-to-use interface for patients and efficient management for healthcare providers.

---

## 🚀 Features

* User registration and authentication
* Browse doctors by specialty
* Book and manage appointments
* View appointment history
* Submit reviews and feedback
* Profile management (update name, phone, etc.)
* Notifications for upcoming appointments

---

## 🛠️ Tech Stack

* **Frontend:** React.js
* **Styling:** CSS / Bootstrap (if applicable)
* **State Management:** React Hooks (useState, useEffect)
* **Routing:** React Router

## ⚙️ Installation & Setup

1. Navigate to the project folder:

   ```bash
   cd grihf-frontend_capstone_starter_code
   ```

2. Install dependencies:

   ```bash
   npm install
   ```

3. Run the application:

   ```bash
   npm start
   ```

4. Build for production:

   ```bash
   npm run build
   ```
